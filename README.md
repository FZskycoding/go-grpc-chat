# GoChatRPC

這是一個使用 Go 語言和 gRPC 實現的即時聊天系統。該專案旨在展示 gRPC 的各種功能特性，並提供一個清晰的學習路徑。

## 專案目標

- 實現一個基於 gRPC 的即時聊天系統
- 展示 gRPC 的不同通訊模式（Unary、Server Streaming、Client Streaming、Bidirectional Streaming）
- 理解 Protocol Buffers 的使用方式
- 學習 Go 語言在實際專案中的應用
